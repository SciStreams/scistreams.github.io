SciStreams

SciStreams aims to help researchers by making their science more accessible through interactive web applications.

Whether you have a research paper, python package or dataset - everything to a certain degree can be turned into an App that you can share with community. At the moment, we are focusing on App developement using Streamlit.

How will new SciStreams come about?

Currently, there are two ways:

1) We see some interesting research paper, python package or dataset and decide to turn it into an interactive web application.
2) You make a suggestion of what would you like to see as a web application.

How to make a suggestion?

Just send us an email with link to a paper, python package, dataset or any other form of scientific research that can be potentially turned into an App. scistreams.info@gmail.com

What happens after SciStreams are published?

SciStreams are going to be made publicly available. You are free to enjoy them, you can also clone GitHub repo and continue expanding them or make a direct contribution to SciStreams.

SciStreams codes are publicly available on our GitHub
